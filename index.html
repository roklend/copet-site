Quickstart

  1. Download and unpack the latest CoPet-vX.Y.Z-win-x64.zip.
  2. Run CoPet.exe.
  3. If onboarding opens, click Apply patch (optional: Backup first).
  4. Open VS Code and use Codex normally.
  5. Expected result: CoPet reacts to thinking, tool, done, error.

How it works

  • CoPet runs a local OTLP HTTP listener on 127.0.0.1:9009.
  • It parses Codex telemetry events (/v1/logs, /v1/traces).
  • An internal state machine maps events to pet states.
  • The UI renderer animates satellite frames and telemetry HUD in real time.
  • Error and done windows are held briefly before returning to idle.
